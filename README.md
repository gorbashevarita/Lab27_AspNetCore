**Список реализованных маршрутов с описанием**: 

Главная страница
```

```

```

    Group = "ИСП-234",

    IsActive = true
});
```
JSON-массив
```

    "РПМ",
    "РМП",
    "ИСРПО",
    "СП"
});
```
JSON с использованием класса
```
app.MapGet("/product/{id}", (int id) => new Product(
    Id: id,
    Name: $"Товар #{id}",
    Price: id * 99.99m,
    InStock: id % 2 == 0
));
```
Главная
```
app.MapGet("/", () => Results.Ok(new
{
    Message = "Добро пожаловать!",
    Version = "1.0",
    Time = DateTime.Now.ToString("HH:mm:ss")
}));
```
Инофрмация о студенте
```
app.MapGet("/me", () => Results.Ok(new
{
    Name = "Иванов Иван",
    Group = "ИСП-234",
    Course = 3,
    Skills = new[] { "C#", "HTML", "CSS", "JS", "ASP.NET" }
}));
```
Калькулятор
```
app.MapGet("/calc/{a}/{b}", (double a, double b) =>
    Results.Ok(new
    {
        A = a,
        B = b,
        Sum = a + b,
        Diff = a - b,
        Mul = a * b,
        Div = b != 0 ? a / b : 0
    }));
```
Машрут не найден
```
app.MapFallback(() => Results.NotFound(new {
    Error = "Маршрут не найден",
    Code = 404
}));
```

**Ответы на вопросы**:

*1.Чем Minimal API отличается от MVC в ASP.NET Core?*

Minimal API — упрощённый подход без контроллеров, MVC — классическая архитектура с контроллерами и соглашениями.

*2. Что произойдёт, если поставить app.MapGet(...) до app.Use(...)?*

Маршруты могут не сработать: middleware нужно регистрировать до маршрутов.

*3.	Почему ASP.NET Core преобразует PascalCase в camelCase при сериализации JSON?*

Для совместимости с JavaScript‑клиентами, где принят camelCase.

*4.	Что означает код ответа HTTP 401? А 404? А 200?*

401 — не авторизован; 404 — ресурс не найден; 200 — успех.

*5.	Чем `dotnet run` отличается от `dotnet watch run`?*

`dotnet run` запускает один раз, `dotnet watch run` перезапускает при изменениях кода.
